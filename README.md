# 🏁 Kiwiplay — 키위 경쟁 게임

브라우저에서 동작하는 정적 경쟁 게임. 1~10명 등록 → 게임 선택 → 경주.

## 게임 종류
- 🏊 **수영 게임** — 45도 기울기 + 위아래 보빙 / 🦈 상어, 💢 다리쥐, 🪼 해파리 부스트
- 🏃 **달리기 게임** — 위로 통통 튀는 모션 / 🍌 바나나, ⚡ 에너지드링크, 🚧 허들
- 🚀 **우주 날기 게임** — 좌우 드리프트 + 회전 / 🕳️ 블랙홀, ☄️ 운석, ✨ 워프
- 🍺 **술 마시기 게임** — 비틀거리는 스태거 모션 / 🍻 원샷, 🤢 토함, 🥂 건배(전원), 😵 필름끊김, 🍤 안주

## 플레이어
- 1~10명 등록
- 이미지 업로드 가능 (선택)
- 미등록 시 이름 첫 글자 기반 컬러 그라디언트 SVG 아바타 자동 생성

## 로컬 실행
순수 정적 파일이라 더블클릭으로도 열리지만, file:// 보안 제약 회피를 위해 간단한 서버 권장:

```sh
# Python
python -m http.server 8080

# Node
npx serve .
```

http://localhost:8080 접속.

## GitHub Pages 배포
1. GitHub 새 리포지토리에 이 폴더 푸시 (`index.html`이 루트에 있어야 함)
2. Settings → Pages → Source: `main` branch / `/(root)`
3. `https://<username>.github.io/<repo>/` 로 접속

## 파일 구조
```
index.html   # 마크업
style.css    # 스타일 + 트랙 테마
script.js    # 게임 엔진, 이벤트, 아바타 생성
```
